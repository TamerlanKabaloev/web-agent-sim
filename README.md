# Agent Tunnel Simulation

This is a web-based agent simulation similar to AnyLogic, built with React, Three.js, and Zustand.

## Features

- **Parametric Tunnel**: Adjust width and length in real-time.
- **Agent Simulation**: Agents spawn and move through the tunnel.
- **Configuration**: Control spawn rate, agent speed, and dimensions.
- **3D Visualization**: View the simulation from any angle.

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the browser at the URL shown (usually `http://localhost:5173`).

## Controls

- **Right Panel**: Adjust simulation parameters.
- **Mouse Left Click + Drag**: Rotate camera.
- **Mouse Right Click + Drag**: Pan camera.
- **Mouse Wheel**: Zoom.
